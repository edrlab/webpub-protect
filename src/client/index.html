Original documents in iframe:

  1. text can be selected
  2. selected text can be copied to clipboard (try in the text box ⇒⇒⇒)
  3. context menu can be displayed with mouse right-click
  4. web inspector / debugger can be opened via context menu ("inspect element")
  5. the original document markup is revealed in the web inspector

Protected documents in iframe:

  1. text cannot be selected
  2. text cannot be copied to clipboard (try in the text box ⇒⇒⇒)
  3. context menu cannot be displayed with mouse right-click
  4. web inspector / debugger cannot be opened via context menu ("inspect element")
  5. the original document markup can still be revealed in the web inspector (using the top-level web browser menu to open the debugging tools, and searching for the iframe in the DOM)
